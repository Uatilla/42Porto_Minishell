**Minishell** is a C project developed as part of the 42 School Common Core Program. The primary objective of this project is to create a program that effectively manages data races and avoid deadlocks. Just as in economics, where resources are finite and must be allocated wisely, programming requires careful management of resourcesâ€”typically dataâ€”that need to be accessed by multiple processes or threads. It is essential to ensure that this data remains accurate and reliable. In this context, mutexes are employed to lock the data, preventing concurrent access and ensuring its integrity.

For more detailed information, look at the [**subject of this project**](https://cdn.intra.42.fr/pdf/pdf/138331/en.subject.pdf).

One of our learnings after building this project is that **everyone can apply mathematics or physics without being a mathematician or physician.**

	- The first time will download and set up the minilibx
 
	- Run ./minishell
